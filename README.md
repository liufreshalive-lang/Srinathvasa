# 🚗 Citroën Sales Consultant Portal

A modern, mobile-first web application designed for **Citroën Sales Consultants** to manage their daily workflow, customer leads, and sales targets efficiently.

## 🌟 Key Features

- **📊 Dynamic Dashboard**: Quick overview of daily leads, pending tasks, and monthly sales performance.
- **👥 Lead Management**: Track customer inquiries, vehicle preferences, and follow-up history.
- **✅ Task Tracking**: Manage daily activities like calls, emails, and upcoming test drives.
- **📄 Quote Creation**: Generate on-road price quotes for different Citroën models and variants.
- **👤 Performance Analytics**: Real-time tracking of targets versus achievements.
- **📱 Responsive Design**: Fully optimized for mobile devices and tablets.

## 🛠️ Built With

- **HTML5**: Semantic structure for better accessibility.
- **CSS3**: Modern layout with custom styling and smooth animations.
- **JavaScript (ES6+)**: Interactive UI transitions and state management.

## 🚀 Deployment

The portal is hosted on **GitHub Pages**.

**Live URL**: `https://liufreshalive-lang.github.io/Srinathvasa/`

## 🖱️ How to Use

1. **Login**: Use the demo credentials to access the portal.
2. **Dashboard**: Navigate through the core features using the bottom navigation bar.
3. **Leads**: Add new leads or view existing customer details.
4. **Tasks**: Check your daily agenda and mark tasks as complete.
5. **Quotes**: Create and preview vehicle quotes.
